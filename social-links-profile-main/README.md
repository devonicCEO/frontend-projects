# Social Links Profile

## English

A compact profile card with social links and contact information. Built with HTML and CSS for easy embedding into personal pages.

Features:

- Profile card layout
- Social icons and link placeholders
- Simple and customizable styles

How to view: open index.html in your browser.

License & Credits: See LICENSE in this folder. Assets and fonts are in the assets directory.
