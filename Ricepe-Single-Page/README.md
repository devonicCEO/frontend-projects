# Ricepe Single Page

## English

A simple, single-page recipe layout showcasing ingredients and steps using HTML and CSS. Open index.html to preview the page in a browser.

Highlights:

- Clean, mobile-first layout
- Readable typography and images
- Easy to adapt for other recipes

License & Credits: See LICENSE in this folder. Fonts and assets are included in the assets directory.
