# FAQ Accordion

## English

A clean, responsive FAQ accordion built with plain HTML, CSS and minimal JavaScript. Open index.html in a browser to view the demo.

Features:

- Responsive layout
- Smooth expand/collapse animations
- Keyboard and accessibility friendly

License & Credits: See the LICENSE file in this folder. Fonts and images are stored in the assets directory.
