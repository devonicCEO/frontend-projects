# Static Portfolio Site

## English

A static portfolio template with multiple pages (index, projects, resume, contact). Built with HTML, CSS and simple JavaScript for interactive elements. Open index.html to view the homepage.

Included pages:

- index.html (home)
- projects.html
- resume.html
- contact.html

Features:

- Multi-page static layout
- Responsive grid for projects
- Lightweight and easy to customize

License & Credits: See LICENSE in this folder. Assets and scripts are in the assets and js directories.

- index.html (главная)

- resume.html
- contact.html
